Technical

The iolla website is dynamically generated by a Kirby 3 engine and Lithe substrate with static deployment. User controlled contrast mode by Piccalilli. Code crafted in BBEdit. SVG pictograms and icons created in Designer. Served to you this day via Github and Cloudflare.

Accessibility

From the outset I have made every effort to make my website easy to use and accessible to everyone. However, the web is constantly evolving which makes this is an ongoing process. If you discover a usability or accessibility issue that I have overlooked then please let me know so that I can rectify this. Thank you for your help.
